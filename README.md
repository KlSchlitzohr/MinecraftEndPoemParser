# MinecraftEndPoemParser

This is an easy way to create Minecraft End Poems.

# How it works

- Create input.txt
- Fill input.txt with Content
- Run Jar File output.json will appear.

# How to write input.txt
```
#Section 1
*Title 1
-Name 1
-Name 2
-Name 3
#Section 2
*Title 1
-Name 1
-Name 2
*Title 2
-Name 1
-Name 2
```
